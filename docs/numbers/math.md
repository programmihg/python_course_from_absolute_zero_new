–ö–æ–≥–∞—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∏—Ç–µ –∞—Ä–∏—Ç–º–µ—Ç–∏—á–Ω–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –Ω–µ —Å–∞ –¥–æ—Å—Ç–∞—Ç—ä—á–Ω–∏, –∏–∑–ø–æ–ª–∑–≤–∞–º–µ –≤–≥—Ä–∞–¥–µ–Ω–∏—è –º–æ–¥—É–ª `math`.

### 1. math.ceil() –∏ math.floor()
* **math.ceil(x)**: –ó–∞–∫—Ä—ä–≥–ª—è –≤–∏–Ω–∞–≥–∏ **–Ω–∞–≥–æ—Ä–µ** –¥–æ –Ω–∞–π-–±–ª–∏–∑–∫–æ—Ç–æ —Ü—è–ª–æ —á–∏—Å–ª–æ (–Ω–∞–ø—Ä. `3.1 -> 4`).
* **math.floor(x)**: –ó–∞–∫—Ä—ä–≥–ª—è –≤–∏–Ω–∞–≥–∏ **–Ω–∞–¥–æ–ª—É** –¥–æ –Ω–∞–π-–±–ª–∏–∑–∫–æ—Ç–æ —Ü—è–ª–æ —á–∏—Å–ª–æ (–Ω–∞–ø—Ä. `3.9 -> 3`).

### 2. –§—É–Ω–∫—Ü–∏—è—Ç–∞ round()
–ó–∞ —Ä–∞–∑–ª–∏–∫–∞ –æ—Ç –º–æ–¥—É–ª–∞ `math`, `round()` –µ –≤–≥—Ä–∞–¥–µ–Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è –∏ —Å–ª–µ–¥–≤–∞ —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ –ø—Ä–∞–≤–∏–ª–æ (Banker's Rounding):
* –¢—è –∑–∞–∫—Ä—ä–≥–ª—è –∫—ä–º –Ω–∞–π-–±–ª–∏–∑–∫–æ—Ç–æ **—á–µ—Ç–Ω–æ** —á–∏—Å–ª–æ, –∞–∫–æ –¥–µ—Å–µ—Ç–∏—á–Ω–∞—Ç–∞ —á–∞—Å—Ç –µ —Ç–æ—á–Ω–æ `.5`.
* –ü—Ä–∏–º–µ—Ä: `round(2.5)` –µ `2`, –Ω–æ `round(3.5)` –µ `4`.

```python
import math

print(math.ceil(2.1))   # 3
print(math.floor(2.9))  # 2
print(round(2.5))       # 2 (—Å–ø–µ—Ü–∏—Ñ–∏–∫–∞ –Ω–∞ Python)
```
