# ⚠️ Често срещани грешки (Python)

### 1. Забравено двоеточие (`:`)
Винаги след `if`, `while` или дефиниция на функция трябва да има двоеточие.
*❌ Грешно:* `if x > 5`
*✅ Правилно:* `if x > 5:`

### 2. Индентация (Интервали)
Python използва интервалите, за да разбере кой код към кой блок принадлежи. Винаги под `if` или `while` трябва да има точно 4 интервала (или един Tab).

### 3. Единично срещу двойно равно
* `=` се използва за **задаване** на стойност (`x = 10`).
* `==` се използва за **сравнение** (`if x == 10:`).
